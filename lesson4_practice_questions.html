CompTIA Security+ • Lesson 4

Practice / Study Questions (Multiple Choice)

Topic: Identity and Access Management
15 Questions
Printable Format

Instructions

Circle or mark the correct option for each question. Each item tests concepts from Lesson 4: Identity and Access Management (IAM), including SSO, federation, access controls, MFA, and AAA protocols.

Questions

  1. Which access control model assigns permissions to job roles instead of individual users?
    1. Discretionary Access Control (DAC)
    2. Role-Based Access Control (RBAC)
    3. Mandatory Access Control (MAC)
    4. Rule-Based Access Control
  2. In a federated sign-in between Azure AD and Salesforce, Azure AD acts as the:
    1. Service Provider
    2. RADIUS Server
    3. Identity Provider
    4. Authorization Server within Salesforce
  3. Which protocol enables secure single sign-on between an identity provider and a service provider for web applications?
    1. OAuth 2.0
    2. SAML
    3. RADIUS
    4. Kerberos
  4. Which statement accurately describes OAuth 2.0?
    1. A protocol used primarily for authentication
    2. An authorization framework that allows resource access without sharing credentials
    3. A replacement for SAML in enterprise SSO
    4. A network protocol for device management
  5. Which option best defines OpenID Connect?
    1. A stand-alone replacement for OAuth 2.0
    2. An identity layer built on top of OAuth 2.0 to add authentication
    3. A version of Kerberos used in Azure AD
    4. A certificate enrollment mechanism
  6. Which step in the SAML authentication flow occurs first when a user attempts to access Salesforce through Azure AD SSO?
    1. The Service Provider redirects the user to the Identity Provider
    2. The Identity Provider validates credentials and issues a signed assertion
    3. The Service Provider validates the assertion and grants access
    4. The user initiates access to the Service Provider application
  7. Which Azure AD control helps enforce sign-in requirements based on location, device compliance, or risk level?
    1. Password Complexity Policy
    2. Conditional Access Policy
    3. Account Lockout Threshold
    4. Session Timeout Setting
  8. Which combination represents true multifactor authentication (MFA)?
    1. Password and Username
    2. Password and Security Question
    3. Smart Card and Authenticator App Code
    4. PIN and Password
  9. Which protocol provides centralized authentication and command authorization for managing network devices?
    1. Kerberos
    2. RADIUS
    3. TACACS+
    4. LDAP
  10. Which access control model evaluates multiple user and environmental attributes to make decisions dynamically?
    1. Role-Based Access Control (RBAC)
    2. Attribute-Based Access Control (ABAC)
    3. Discretionary Access Control (DAC)
    4. Mandatory Access Control (MAC)
  11. Which principle states that users should be granted only the minimum permissions required to perform their job duties?
    1. Separation of Duties
    2. Least Privilege
    3. Defense in Depth
    4. Zero Trust
  12. In which access control model can resource owners determine who can access their objects?
    1. Discretionary Access Control (DAC)
    2. Mandatory Access Control (MAC)
    3. Role-Based Access Control (RBAC)
    4. Attribute-Based Access Control (ABAC)
  13. Which method provides the strongest, most phishing-resistant second factor for administrator accounts?
    1. SMS One-Time Code
    2. Email Approval Link
    3. Authenticator App Code (TOTP)
    4. FIDO2 Hardware Security Key
  14. During an access review, some users still have rights to an app after changing departments. Which control corrects this?
    1. Emergency Break-Glass Accounts
    2. Role Recertification and Access Reviews
    3. Shared Administrative Accounts
    4. Extended Password Expiration Policy
  15. Which step should occur immediately after an employee leaves the company to maintain security?
    1. Archive the user’s mailbox
    2. Disable or delete the account and revoke access tokens
    3. Wait 30 days before removing the user
    4. Reset the password and keep the account active

Answer Key

  1. B Role-Based Access Control (RBAC)
  2. C Identity Provider
  3. B SAML
  4. B Authorization framework for delegated access
  5. B Identity layer that adds authentication to OAuth 2.0
  6. D User initiates access to the Service Provider
  7. B Conditional Access Policy
  8. C Smart Card and Authenticator App Code
  9. C TACACS+
  10. B Attribute-Based Access Control (ABAC)
  11. B Least Privilege
  12. A Discretionary Access Control (DAC)
  13. D FIDO2 Hardware Security Key
  14. B Role Recertification and Access Reviews
  15. B Disable or delete the account and revoke tokens
